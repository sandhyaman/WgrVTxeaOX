```markdown
# 项目简介

本系统是一款基于Python和TensorFlow框架的深度学习驱动的网络舆情监控解决方案，主要用于微博、论坛、新闻等多源文本的情感极性分析。

## 核心功能

- **智能文本分类**：支持CNN/LSTM双模型并行分析，进行结果对比验证。
- **多维度可视化**：
  - CNN/LSTM准确率对比柱状图。
- **全流程管理后台**：
  - 文本数据管理（增删改查）。
  - 模型结果人工校正。
  - 多用户管理。
- **实时分析API**：输入文本即时返回情感分类结果。

## 适用场景

- 课程设计
- 毕业设计
- 企业舆情监控
- 品牌口碑分析

## 技术特点

- **双模型融合架构**：CNN捕捉局部特征 + LSTM建模上下文依赖。
- **预训练词向量**：支持Word2Vec/GloVe嵌入，提升语义理解。
- **动态可视化**：Echarts实现交互式数据分析看板。
- **工业级部署**：提供Docker容器化方案，支持高并发请求。
```
